### 💻 Hướng dẫn cài đặt và khởi chạy (Installation & Setup)

#### 1. Yêu cầu hệ thống
* Hệ điều hành: Windows (Khuyến nghị).
* Ngôn ngữ: Python 3.10+.
* Thư viện bổ sung: `customtkinter` (Giao diện hiện đại).

#### 2. Cài đặt thư viện
Mở Terminal/Command Prompt tại thư mục dự án và chạy lệnh:
```bash
pip install customtkinter
